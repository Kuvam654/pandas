# pandas
Create a folder in a directory (For example C drive) --> Right click + Shift --> Open using Powershell 

Install the following

1. Pandas
```
pip install pandas
```
```
pip install --upgrade pandas
```

2. Numpy
```
pip install numpy
```

3. Jupyter Notebook
```
pip install jupyter
```

4. Open Jupyter Notebook
```
jupyter notebook
```
